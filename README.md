# Vite, Vue 3, Typescript and Tailwind CSS starter

Basic Vue 3 starter with Typescript, Eslint, Prettier and Tailwind CSS

## Features

- Vue 3
- TypeScript 5
- Pinia store
- Routing using [vue-router 4](https://router.vuejs.org/)
- Tailwind CSS
- ESLint
- Prettier
- Using newest `script setup` syntax
